ZenCNC for PCB's at FamiLab

The first project I worked on when I became a member of FamiLab. I figured out how to setup one of their smaller CNC machines to cut out printed circuit boards using pcb-gcode for eaglecad and a very handy utility I found called autoleveller. If you would like more information go check out my posts on the familab forums My user name is knkp. There is also a video of autoleveller in action - finding all height variations of a pcb.



Homemade IRC

This is a little project I worked on when I was first trying to understand posix style message que's. A friend of mine who enjoy's working on linux boxes with me from time to time found that the method available for communication (specifically the wall command) can be very disruptive when working remotely on a server. Thus I began devicing this little gadget which utlizes message que's to allow multiple user's connected to a server to communicate without interrupting the other's work. It is a work in progress and should probably be refactored from the ground up. But it was a good introduction to posix, linux c code, and emacs. You can take a look at this project here.

Windows Background Launcher

At one point I had several laptops running different versions of linux that I worked on most of the time. Unfortunately, they were stolen and I purchased a new machine from best buy with windows 8.1. I needed to keep the windows operating system for certain software utilities that were not available in linux but I also missed the ease of navigation and quick key-commands too launch programs quickly without having to navigate a slow GUI interface. Because of this limitation I decided to write a tool that ran in the background of my windows box and based on a text file that I setup, it can launch various programs with just a quick key-shortcut. There are probably alternative's to this, but I enjoy writing tools for myself if I have the time. The details of this project can be found here



Simple Home Automation with Raspberry Pi

This was my first project with the raspberry pi. I often had difficulty getting up in the mornings and sometime's an alarm clock by itself just wouldn't do it. So I used a raspberry pi and connected it to an arduino that had a relay shield mounted on top of it which then ran via cut extension cable into the back of my light switch and to a fan and tv plugs that would turn on and off depending on the time of day. I made a quick youtube video of the project before it was disassembled for other hacking experiments
